# Tests Folder

This folder is for temporary test files during development.

**Usage:**
- Create test files here as needed during development
- Clean up test files once they are no longer needed
- Do not commit large test artifacts

**Example:**
```bash
# Run a quick test
python AI_CTA_Stroke/tests/my_test.py

# Clean up when done
rm AI_CTA_Stroke/tests/my_test.py
```
