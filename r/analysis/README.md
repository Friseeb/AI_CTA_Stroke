# R Statistical Analysis

Statistical modeling and reporting:

- `cox_models.R` - Cox proportional hazards for recurrence/MACE
- `calibration.R` - Calibration plots and Brier scores
- `discrimination.R` - C-index, ΔC-index, NRI, IDI
- `sgba_tables.R` - Sex-stratified analyses (SGBA compliance)
- `survival_plots.R` - Kaplan-Meier curves
- `model_comparison.R` - Compare AI-enhanced vs clinical-only models
