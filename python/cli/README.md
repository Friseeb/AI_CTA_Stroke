# CLI Entrypoints

Command-line tools for running pipelines:

- `bids_convert.py` - Convert DICOM to BIDS format
- `train_resnet.py` - Train 3D ResNet for plaque segmentation
- `infer_resnet.py` - Run inference on new CTA scans
- `calc_metrics.py` - Calculate plaque burden and calcium scores
- `export_results.py` - Export predictions to REDCap format
